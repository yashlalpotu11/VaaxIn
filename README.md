## Prerequisites

- node >=11.0.0
- npm >=6.0
- yarn >=1.16.0

## Installation

```bash
   # git clone https://github.com/yashlalpotu11/VaaxIn.git
   cd VaaxIn
	
   npm install
   npm start    # Yes, ONE command for server AND client development!
	
```
Go To **http://localhost:3000/**
